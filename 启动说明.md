# 校园失物招领系统 - 快速启动指南

## 你只需要安装两样东西

1. **JDK 17**（必须）
2. **MySQL 8.0**（必须）

> Redis、RabbitMQ、Elasticsearch、Python 都是**可选的**，不装也能正常运行！

---

## 三步启动

### 第一步：创建数据库

打开 MySQL，执行：

```sql
CREATE DATABASE lost_found DEFAULT CHARACTER SET utf8mb4;
```

然后执行项目里的三个 SQL 脚本：

```
src/main/resources/sql/init.sql          -- 基础表（6张）
src/main/resources/sql/migration_v2.sql   -- 升级表（7张）
sql/demo-data.sql                        -- 测试数据（13表完整数据，便于演示）
```

或者直接双击运行 `init-db.bat`

### 第二步：改数据库密码

打开 `src/main/resources/application.yml`，第 10 行：

```yaml
password: root    # 改成你自己的MySQL密码
```

### 第三步：启动

双击 `start.bat`，或命令行执行：

```
mvn spring-boot:run
```

等 15 秒左右，看到 `Started LostFoundApplication` 就成功了。

浏览器打开：http://localhost:8080

## 常见问题

**Q: 启动报错 "Communications link failure"**
A: MySQL 没启动，或者密码不对。检查 MySQL 服务和 application.yml 里的密码。

**Q: 页面打不开**
A: 等控制台出现 `Started LostFoundApplication` 再访问，启动需要15秒左右。

**Q: 端口8080被占用**
A: 先关掉占用的程序，或者改 application.yml 第2行的 `port: 8080` 为其他端口。

**Q: 上传图片失败**
A: 图片自动保存到项目内 `src/main/webapp/upload/` 目录，确保项目目录有写入权限即可。
